Swift extensions from Siberia with love!

## Requirements

Swift 4.0, iOS version >= 10.0

## Installation

SiberianSwift is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "SiberianSwift"
```
